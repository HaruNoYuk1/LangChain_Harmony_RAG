 挂载卸载事件指组件从组件树上挂载、卸载时触发的事件。 
说明
从API Version 7开始支持。后续版本如有新增内容，则采用上角标单独标记该内容的起始版本。 
 事件 名称  支持冒泡  功能描述    onAppear(event: () => void)  否  组件挂载显示时触发此回调。 从API version 9开始，该接口支持在ArkTS卡片中使用。   onDisAppear(event: () => void)  否  组件卸载消失时触发此回调。 从API version 9开始，该接口支持在ArkTS卡片中使用。      示例已复制// xxx.etsimport promptAction from '@ohos.promptAction'
