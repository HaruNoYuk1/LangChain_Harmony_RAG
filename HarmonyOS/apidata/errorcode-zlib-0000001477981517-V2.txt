 900001 传入的源文件错误错误信息 The input source file is invalid. 错误描述 当调用compress或decompress接口时，传入源文件无效。 可能原因 当调用compress接口时，传入的待压缩的文件不存在；当调用decompress接口时，传入的待解压缩的文件不存在。 处理步骤 检查源文件是否存在。检查待压缩的文件路径是否存在，并且路径是否在正确的沙箱路径下。  900002 传入的目标文件错误错误信息 The input destination file is invalid. 错误描述 当调用compress或decompress接口时，传入目标文件无效。 可能原因 当调用compress接口时，传入的目标文件路径无效，如不存在的沙箱路径。当调用decompress接口时，传入的目标目录不存在。 处理步骤 检查目标文件路径是否正确，不正确填写正确的沙箱路径。检查目标文件目录是否存在，不存在新建这个路径。   上一篇 包管理子系统通用错误码 下一篇 公共事件与通知 以上内容对您是否有帮助？毫无帮助帮助不大一般很好非常好意见反馈
