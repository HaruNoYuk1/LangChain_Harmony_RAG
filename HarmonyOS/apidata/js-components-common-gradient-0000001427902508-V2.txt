 组件普遍支持的在style或css中设置的 可以平稳过渡两个或多个指定的颜色。 开发框架支持线性渐变 (linear-gradient)和重复线性渐变 (repeating-linear-gradient)两种渐变效果。 线性渐变/重复线性渐变使用渐变样式，需要定义过渡方向和过渡颜色。  过渡方向通过direction或者angle指定过渡方向。 direction：进行方向渐变。 angle：进行角度渐变。  已复制background: linear-gradient(direction/angle, color, color, ...);background: repeating-linear-gradient(direction/angle, color, color, ...);  过渡颜色支持以下四种方式：#ff0000、#ffff0000、rgb(255, 0, 0)、rgba(255, 0, 0, 1)，需要指定至少两种颜色。 参数：  名称 类型 默认值 必填 描述   direction to <side-or-corner>  <side-or-corner> = [left | right] | [top | bottom] to bottom (由上到下渐变) 否 指定过渡方向，如：to left (从右向左渐变)  ；或者 to bottom right (从左上角到右下角)。   angle <deg> 180deg 否 指定过渡方向，以元素几何中心为坐标原点，水平方向为X轴，angle指定了渐变线与Y轴的夹角(顺时针方向)。  color <color> [<length>|<percentage>] - 是 定义使用渐变样式区域内颜色的渐变效果。    示例：  默认渐变方向为从上向下渐变。 已复制#gradient {  height: 300px;  width: 600px;  /* 从顶部开始向底部由红色向绿色渐变 */  background: linear-gradient(red, #00ff00);} ​   45度夹角渐变。 已复制/* 45度夹角，从红色渐变到绿色 */  background: linear-gradient(45deg, rgb(255,0,0),rgb(0, 255, 0)); ​   设置方向从左向右渐变。 已复制/* 从左向右渐变，在距离左边90px和距离左边360px (600*0.6) 之间270px宽度形成渐变 */background: linear-gradient(to right, rgb(255,0,0) 90px, rgb(0, 255, 0) 60%);   重复渐变。 已复制/* 从左向右重复渐变，重复渐变区域30px（60-30）透明度0.5 */background: repeating-linear-gradient(to right, rgba(255, 255, 0, 1) 30px,rgba(0, 0, 255, .5) 60px);     上一篇 动画样式 下一篇 转场样式 以上内容对您是否有帮助？毫无帮助帮助不大一般很好非常好意见反馈
