 用于触发长按手势事件，触发长按手势的最少手指数为1，最短长按时间为500毫秒。 
说明
  从API Version 7开始支持。后续版本如有新增内容，则采用上角标单独标记该内容的起始版本。 
 接口LongPressGesture(value?: { fingers?: number, repeat?: boolean, duration?: number }) 参数： 参数名称 参数类型 必填 参数描述   fingers number 否 触发长按的最少手指数，最小为1指， 最大取值为10指。 默认值：1   repeat boolean 否 是否连续触发事件回调。 默认值：false   duration number 否 触发长按的最短时间，单位为毫秒（ms）。 默认值：500      事件  名称 功能描述 onAction(event:(event?: GestureEvent) => void) LongPress手势识别成功回调。  onActionEnd(event:(event?: GestureEvent) => void) LongPress手势识别成功，手指抬起后触发回调。  onActionCancel(event: () => void) LongPress手势识别成功，接收到触摸取消事件触发回调。     示例已复制// xxx.ets@Entry@Componentstruct LongPressGestureExample {  @State count: number = 0
